The Platformer Game Kit is a FREE Unity plugin which provides an extensible framework for creating 2D platformer games using Animancer.

It uses sprites from the Animated Character Pack made by Szadi Art as examples and the Set Up Instructions explain how easy it is to import custom sprites or 3D models and make animations using the Animancer Tools or Unity's inbuilt tools.

This plugin (and documentation) is a work in progress and has not yet been publicly released. Suggestions and feedback are welcome.

Getting Started

  1. This system requires Unity 2019.4+ and is safe to use in an existing project.
  2. Download and import Animancer Lite or Animancer Pro v7.0+.
  3. Download and import the Platformer Game Kit.
  4. Read the Documentation for more information about how to set things up and how everything works.

Character Features

  • Step up and down stairs
  • Moving Platforms
  • Jump (hold to go higher), Wall Slide, Wall Jump, Air Jump, Land
  • Attack Combo, Attack Up, Throw Projectile
  • Enemies that Patrol, Stand Still, or Fly
  • Introductions

Other Features

  • Hit Boxes (with custom editor), Damage, Health, Destructible Blocks
  • Camera Shake
