Art

The Platformer Game Kit uses sprites from the Animated Character Pack made by Szadi Art as examples. The package includes those assets fully configured and you can use them in your own projects if you want, but this section explains the full set up process so that you can import your own custom sprites.

Import Importing images and configuring their sprite settings
Organization Giving the sprites proper names and compiling them into a Sprite Atlas
